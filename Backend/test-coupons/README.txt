Program generates unique coupon codes
- 10 Characters long starting with 'MQ'
- Alpha-numeric (mixed with lowercase and uppercase letters)

Run program
===========
$ node generateCoupons.js [option:number of coupon codes greater than 0]

e.g.
$ node generateCoupons.js 10000

Notes:
- Coupon codes are generated and written to file 'section2'
- 250 is default coupon codes
